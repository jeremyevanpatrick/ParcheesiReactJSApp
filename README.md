# Parcheesi React App

A browser-based implementation of **Parcheesi** built with **React 17**. Players move pieces around the board according to dice rolls, racing to get all four pieces home before their opponents.

---

## Live Demo

Hosted on GitHub Pages:
👉 [https://jeremyevanpatrick.github.io/ParcheesiReactJSApp](https://jeremyevanpatrick.github.io/ParcheesiReactJSApp)

---

## About the Game

Players roll two dice each turn and move their pieces along a shared track, using strategy to send opponents' pieces back to start, form blockades, and navigate their own pieces safely home. The first player to move all four pieces to the center home space wins.

---

## Features

- Interactive Parcheesi board rendered in React
- Dice rolling mechanics with valid move logic
- Piece movement and capture rules
- Multi-player turn management
